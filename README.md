## Installation

You need node, npm, bower ember-cli console applications
```sh
npm install bower -g
npm install ember-cli -g
```

Build project
```sh
./depoy.sh
```

Start http server
```sh
node back/bin/www
```

**Note:** Don't forget about MongoDB database
